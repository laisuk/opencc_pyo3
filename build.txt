create module folder in project root /opencc_pyo3 same name as stated in cargo.toml:
[lib]
name = "opencc_pyo3"

create pyproject.toml

prepare files in module folder:
__init__.py
opencc_pyo3.pyi

prepare .venv if needed
maturin develop -r
(This create pyd into opencc_pyo3 folder, need venv, if any old pyd in the folder, delete it first)

Exit .venv to Windows system Python env (Deactivate) (Optional)
maturin build -r
(whl created in ./target/wheels/

Install Package (WHL file name may vary)
Windows:
pip install .\target\wheels\opencc_pyo3-0.8.2-cp38-abi3-win_amd64.whl --force-reinstall

Linux:
pip install ./target/wheels/opencc_pyo3-0.8.2-cp312-cp312-manylinux_2_34_x86_64.whl --break-system-packages --force-reinstall
